# Archive Folder

This folder contains **old/unused files** that are no longer part of the main application.

## Files Archived:

### Old Versions:
- `enhanced_chat_interface_0(old).py` - Original chat interface (replaced)
- `enhanced_chat_interface_1.py` - Version 1 (replaced)
- `enhanced_chat_interface_2.py` - Version 2 (replaced by enhanced_chat_interface.py)
- `research_paper_assistant(old).py` - Old version of main app (replaced)

### Alternative/Standalone Apps:
- `app_full.py` - Alternative full-featured app (not used in production)
- `citation_analyzer_comparison_app.py` - Standalone comparison tool for testing different citation analyzers
- `citation_analyzer_semantic.py` - Early semantic analyzer (replaced by hybrid version)

## Why Archived?

These files were archived to:
- Keep the main directory clean and organized
- Preserve old code for reference
- Prevent confusion about which files are actively used
- Maintain project history

## Restoration:

If you need to restore any file, simply copy it back to the main directory:
```bash
cp archive/<filename> .
```
